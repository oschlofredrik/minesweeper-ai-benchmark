Design Philosophy Comparison

Current Design (Terminal/Brutalist)

  • Inspiration: Terminal interfaces, brutalist web
  • Colors: Black background, green accents
  • Typography: Monospace fonts
  • Layout: Dense, information-heavy
  • Feel: Technical, hacker aesthetic

Good for: Technical users who appreciate terminal aesthetics and dense information display.

Dieter Rams Design

  • Inspiration: Braun products, "Less but better"
  • Colors: Monochromatic, high contrast
  • Typography: Clean sans-serif, consistent hierarchy
  • Layout: Grid-based, generous whitespace
  • Feel: Professional, timeless, functional

Good for: Professional environments, focusing on clarity and usability over aesthetics.

Current: Compact, multiple columns

Visual Style

Current: Terminal green-on-black

Rams: Black-on-white with grays

Implementation Notes

The Dieter Rams design uses:

  • Custom CSS with CSS variables for consistency
  • 8px grid system for perfect alignment
  • Type scale based on perfect fourth ratio (1.333)
  • Responsive grid that collapses on mobile
  • Dark mode support via prefers-color-scheme
  • No external dependencies - pure CSS

To switch designs: Update the src/api/main.py file to serve index-rams.html instead of index.html as the root page.
